# SAFE Demo Assets

Use these assets for pilot demos and screenshots:

- `docs/DEMO_GUIDE.md` for the local demo script.
- `docs/SCREENSHOT_GUIDE.md` for required screenshots.
- `README.md`, `INSTALL.md`, and `DEPLOY.md` for setup context.
- `openapi/safe-api.yaml` for API walkthroughs.

Demo checklist:

- Start with `python app.py`.
- Open `/client/overview` for the customer-facing executive view.
- Open `/soc` for analyst operations.
- Open `/api/openapi.yaml` for API documentation.
- Keep NetGuard references limited to legacy compatibility explanations.
