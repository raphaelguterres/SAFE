# SAFE Screenshot Guide

Capture screenshots that show SAFE as an Enterprise XDR / EDR Platform:

1. Client overview at `/client/overview`.
2. SOC overview at `/soc`.
3. Incidents queue at `/soc/incidents`.
4. Host triage at `/soc/hosts`.
5. Admin observability at `/performance-live` or `/observability`.
6. OpenAPI documentation from `openapi/safe-api.yaml`.

Use a clean demo tenant, avoid exposing real hostnames or tokens, and verify the SAFE brand is visible in the first viewport.
