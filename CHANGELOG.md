# Changelog

## v0.1.0-enterprise-preview - 2026-05-11

### Added

- SAFE rebrand and enterprise console direction.
- Windows-first SAFE Agent foundation.
- XDR ingestion, detection, correlation, Kill Chain, MITRE, and risk engines.
- Enterprise protection layer with response policy and guarded executor.
- Agent trust, replay protection, action signing, config validation, and API guard.
- SOC experience, case management, approvals, hunts, metrics, and executive view.
- Operational reliability core with event bus, queues, workers, health, recovery, and SAFE Mode.
- AI-assisted SOC context, explainability, prioritization, and playbook recommendations.
- Security data platform with canonical telemetry, normalization, enrichment, lineage, replay, QA, graph, retention, detection packs, and search.
- Production readiness scripts, demo seed, installer docs, CI release workflow, Docker packaging, release notes, and health reporting.

### Security

- Maintains fail-closed posture for secrets and dangerous response actions.
- Keeps RBAC, CSRF, tenant isolation, audit, replay protection, and action signing in scope.

### Quality

- Pytest and pentest audit gates are the required release baseline.
